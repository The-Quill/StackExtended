StackExtended

A list of userscripts and extensions for Stack Exchange

View On GitHub
Downloads:
  • ZIP
  • TAR

    Main site

    • Stack Overflow Extras by ᔕᖺᘎᕊ

      Adds a whole lot of extra functionality and options to Stack Exchange. Full list here

    • Sidebar comments by Normal Human

      Displays recent comments in the sidebar.

    • AutoReviewComments by Benjol

      Lets you keep lists of prefill comments to use.

    • Freehand Circles Drawing Tool by ᔕᖺᘎᕊ

      Red circles! Drawing stuff!

    • Threaded comments by balpha

      Threads comments so you can see the logical flow better.

    Review

    • Review+ Page by Normal Human

      Moves review items to the sidebar for better clarity and displays a list of potentially VLQ or problem questions.

    • Review Stalker Reloaded by Nathan Tuggy

      Lets you keep one tab of /review open that jumps around to all the other sites that you can review on.

    • Audit Detector by Normal Human

      Shows you when you're getting audited.

    Chat

    • Porkchat by Fredley

      Chat plugin containing Dark Theme, Keyboard Shortcuts and Rebecca Black + Image protection

    • Star Image Unthumbnailer by Madara Uchiha

      Unthumbnails images in the star wall

    • Chatception by KeyboardFire

      Lets you chat on multiple chat rooms from one room.

    • Sevichat by KeyboardFire

      Adds Vi-like commands to chat

    • SE Chat Modifications by Tim Stone

      Adds / like commands to chat

    • SE Chat Oneboxer by Quill

      Oneboxes a variety of links, like Instagram, Coding Horror, GitHub.

    • Memer by Ismael Miguel

      Provides a tooltip for terminology specific to sites

    • Badge Oneboxer by Quill

      Oneboxes [badge:whatever] like the tag oneboxes.

    Site specific

    PPCG

    • Site design by vihanb

      Redesign of the PPCG site with some added functionality.

    • Sandbox Viewer by vihanb

      Lets you view the new "Sandbox for Proposed Questions" proposals in a nice interface and from any page in the site.
